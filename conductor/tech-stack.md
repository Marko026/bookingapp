# Technology Stack

## Core Frameworks
- **Next.js 16 (App Router):** Primary application framework.
- **React 19:** UI library utilizing the latest features.
- **TypeScript:** Ensuring strict type safety across the application.

## Data & Database
- **Supabase (PostgreSQL):** Backend-as-a-service for database and authentication.
- **Drizzle ORM:** TypeScript ORM for type-safe database operations.
- **Postgres.js:** Performant PostgreSQL client for Node.js.

## UI & Styling
- **Tailwind CSS:** Utility-first CSS framework for rapid styling.
- **Shadcn UI:** Reusable component library built with Radix UI and Tailwind.
- **Framer Motion:** Animation library for interactive UI elements.
- **Lucide React:** Icon set for consistent visual language.
- **Lenis:** Smooth scrolling library.

## Form Handling & Validation
- **React Hook Form:** Efficient form management.
- **Zod:** TypeScript-first schema validation for forms and environment variables.
- **next-safe-action:** Type-safe Server Actions.

## Tooling & Infrastructure
- **Biome:** Unified toolchain for linting and formatting (replaces ESLint/Prettier).
- **next-intl:** Internationalization support.
- **Leaflet / React Leaflet:** Interactive maps for apartment locations.
