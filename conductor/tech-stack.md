# Tech Stack: Sanctuary Booking App

## Core Framework & Language
- **Next.js 16 (App Router):** The primary framework for the application.
- **TypeScript:** Ensuring type safety across the codebase.
- **React 19:** Utilizing the latest React features and improvements.

## Frontend & Styling
- **Tailwind CSS:** For utility-first styling.
- **shadcn/ui:** For high-quality, accessible UI components.
- **Framer Motion 12:** For smooth animations.
- **Lenis:** For high-quality smooth scrolling.
- **Lucide React:** For icons.
- **React Leaflet:** For interactive maps.

## Backend & Database
- **Supabase (PostgreSQL):** The primary database and backend service.
- **Drizzle ORM:** A type-safe ORM for interacting with the database.
- **Resend:** For handling transactional emails (bookings, notifications).

## Internationalization & Content
- **next-intl:** For managing localized routes and translations.
- **Tiptap:** Headless rich-text editor for content management.

## Quality & Tooling
- **Biome:** A unified tool for linting and formatting (replacing ESLint and Prettier).
- **Vitest:** For unit and integration testing.
- **Zod:** For schema-based data validation.
- **React Hook Form:** For efficient form management.
